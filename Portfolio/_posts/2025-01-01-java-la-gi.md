## Java là gì?
Java là ngôn ngữ lập trình hướng đối tượng, được sử dụng rộng rãi
trong phát triển phần mềm doanh nghiệp, web và mobile.

## Ưu điểm của Java
- Đa nền tảng (Write Once, Run Anywhere)
- Bảo mật cao
- Cộng đồng lớn

## Ứng dụng thực tế
- Backend server
- Android
- Hệ thống doanh nghiệp
