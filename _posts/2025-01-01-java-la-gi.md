## Java là gì?

Java là một ngôn ngữ lập trình hướng đối tượng, mạnh mẽ và độc lập nền tảng.

### Đặc điểm nổi bật
- Lập trình hướng đối tượng (OOP)
- Chạy trên JVM
- Được dùng rộng rãi trong doanh nghiệp

### Ứng dụng thực tế
- Backend Web
- Android
- Hệ thống doanh nghiệp
