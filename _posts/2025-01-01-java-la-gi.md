## Java là gì?

Java là một ngôn ngữ lập trình hướng đối tượng, được phát triển bởi Sun Microsystems (nay thuộc Oracle). Java nổi tiếng với triết lý **“Write Once, Run Anywhere”**, cho phép chương trình chạy trên nhiều nền tảng khác nhau mà không cần chỉnh sửa mã nguồn.

## Đặc điểm nổi bật của Java
- Độc lập nền tảng nhờ JVM
- Hướng đối tượng hoàn toàn
- Bảo mật cao
- Thư viện phong phú

## Ứng dụng của Java
Java được sử dụng rộng rãi trong:
- Phát triển Backend Web
- Ứng dụng Android
- Hệ thống doanh nghiệp
- Hệ thống ngân hàng

## Kết luận
Java là lựa chọn phù hợp cho người mới học lập trình và các hệ thống lớn yêu cầu tính ổn định cao.
