## Collection Framework là gì?
Collection Framework cung cấp các cấu trúc dữ liệu chuẩn như List, Set, Map.

## Các interface chính
- List
- Set
- Map
- Queue

## Lợi ích
- Tối ưu hiệu năng
- Giảm thời gian phát triển

## Kết luận
Collection Framework là công cụ không thể thiếu khi lập trình Java.
